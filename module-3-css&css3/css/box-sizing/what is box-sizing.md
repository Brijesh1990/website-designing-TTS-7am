# what is box-sizing ? 
  box sizing used in css 
  box sizing is create a size of any box using its margin and padding 

## what is margin  ? 
   margin provides space after border from four side in any webpage that is called margin.

   or

   space after border any any webpage that is called margin

   **css margin**

   ```
   margin:25px;
   margin: 2px 15px;
   margin: 0px 5px 12px 20px;
   margin-top:25px;
   margin-left:25px;
   margin-right:25px;
   margin-bottom:25px;

   ```

## what is padding  ? 
   padding provides space after border and between content 
   i.e called padding
   or
   space after border and between content in any webpage that is called padding.

   **css padding**

   ```
   padding:25px;
   padding: 2px 15px;
   padding: 0px 5px 12px 20px;
   padding-top:25px;
   padding-left:25px;
   padding-right:25px;
   padding-bottom:25px;

   ```
